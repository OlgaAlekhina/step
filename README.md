#  Как запустить проект на своем локальном сервере через IDE PyCharm:
clone this repo
```
git clone 
```
create virtual env
```
python -m venv venv
```
activate created venv
```
venv\scripts\activate
```
install requirements
```
pip install -r requirements.txt
```
run server
```
python manage.py runserver
```
